# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-02-05

### Added
- Initial release of Butterfly CLI
- Login/logout authentication with API keys
- Backup creation and listing
- Backup selection and management
- Diff comparison between backups
- Export to Terraform format
- Export to Git repositories
- Configuration management
- Watch mode for continuous monitoring
- JSON output format support
- Environment variable configuration
- Interactive command-line interface
- Homebrew installation support
- npm installation support
